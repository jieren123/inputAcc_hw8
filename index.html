About this app

This is a unique one-button interface for a calculator application. The following sections provide instructions on how to use the app, information about the tests that have been conducted, and an outline of some of the results and insights by testing and experimentation.

  • Instructions for using the app: The navigation of the calculator's buttons is automated, continually moving through each button in a predefined order. To 'click' or select the currently highlighted button, press the 'a' key on your keyboard.
  • Test results: The tests involved users of different speed with calculator interfaces attempting to solve some calculated and reaction problems.
  • Other insights: Although the one-button interface had a longer average completion time and higher error rate, especially for users who were familiar with standard calculators, it is useful in allowing all users to complete the tasks. This suggests that with practice and familiarity, users could become proficient in using this interface. The one-button interface could be particularly useful for users with accessibility needs or for applications on devices with extremely limited input options.
